      Windows Native Gamers Tools

Description: Tools for pacing, scaling, shading, MIDI rendering and compatibility 

Notes:  
• Tools in this paddock are path‑specific 
• Scaling tools (ShaderGlass, Lossless Scaling, SRWE) → picture only
• Audio tools (VirtualMIDISynth, SoundFonts) → MIDI only
• Wrappers (dgVoodoo2, DXWnd, DDrawCompat) → compatibility only
• Patchers / Shims (Lame Patcher, dinputto8) → compatibility only
• Codecs (K-Lite Codec Pack) → Video and sound playback only
• Lossless Scaling ↔ ShaderGlass do not work well together; choose one path
• SRWE is the primary Windows‑side window manager and scaler
  → It moves offscreen windows, overrides always‑on‑top conflicts, and is required for ShaderGlass custom scaling
• RTSS is only used as an external framerate cap when driver‑level or in‑game caps are not adequate

⎯(ᶜᵒʷ•ᵔ)⎯─◐◑◒◓─ LOSSLESS SCALING
Description: Lossless Scaling settings optimized for clean, stable scaling and LSFG frame generation 

Notes:
• Driver-level frame limiters (NVIDIA/AMD) are unreliable with LSFG
• Does not work well with Shaderglass

• Frame Generations Type: LSFG 3.1  
• Mode: Fixed
• Multiplier (base FPS × multiplier ≈ refresh rate - safety margin)
  → 30 FPS base:
    • 7.8× → ~235 FPS @ 240 Hz
    • 3.9× → ~117 FPS @ 120 Hz
    • 1.9× →  ~57 FPS @  60 Hz
  → 60 FPS base:
    • 3.9× → ~235 FPS @ 240 Hz
    • 1.9× → ~117 FPS @ 120 Hz
• Flow Scale: 30% **preserve clarity** 
• Performance: ON **preserve clarity**  
• Capture API: DXGI **WGC for compatibility** 
• Queue target: 0 **1 for compatibility**
• Clip cursor: Off **Optional**
• Adjust cursor speed: Off **Optional**
• Scaling Type: Sharp Bilinear **clean pixel edges**  
• Rendering Sync Mode: Default **vsync for troubleshooting**
• Max frame latency: 2 **general default**
 • G-Sync support: ON
  
  
⎯(ᶜᵒʷ•ᵔ)⎯─◐◑◒◓─ RTSS (RivaTuner Statistics Server)
Description: External framerate limiter for clean, stable base framerates in games
→ Launch RTSS
→ Add → Game Process
→ Set “Framerate limit” to base FPS  
  
  
⎯(ᶜᵒʷ•ᵔ)⎯─◐◑◒◓─ SHADERGLASS
Description: Applies .slang shaders in real time. **not compatible with Lossless Scaling.**
→ Launch ShaderGlass → Input → Window → select your game  
→ Input → Pixel Size → x1
→ Shader → Choose From Library / Import Custom → choose shader 
→ Output → Scale →  Scaling % **for custom scaling select FREE and use SRWE to adjust Shaderglass window**
  • Press 'm' to toggle menu on/off, useful when scaling with SRWE
→ Performance tweaks **Undo if tearing presents in game**
  • Processing → Advanced → Use Flip Mode: Enabled; Allow Tearing: Enabled; Max Capture Rate: Enabled
  • Processing → FPS → 100% (VSync)
  • Hide menu bar in Shaderglass (Shortcut M) → Processing → Toggle Menu
→ Shader suggestions:
  • RetroArch Library/pixel-art-scaling/sharp-bilinear **purity and edge integrity**
  • misc/simple_color_control **faithfully enhanced color depth**
  
  
⎯(ᶜᵒʷ•ᵔ)⎯─◐◑◒◓─ SRWE (SIMPLE RUNTIME WINDOW EDITOR)
Description: Tool for resizing, repositioning, and border‑taming stubborn apps and games.
→ Launch SRWE → Select Running Application
  • Select Shaderglass.exe when using Shaderglass
→ Window position and size **Scale to 4K resolution at 4x3 aspect ratio** 
  **Press ENTER after entering each value or it won't apply**
  • Shaderglass Config **primary path**:
    → Y: -31; Width: 2896; Height: 2199 **press ENTER after each value**
  • 86Box Config **use only when not using ShaderGlass or LS**
    → Y: -53; Width: 2961; Height: 2221 **press ENTER after each value** 
→ Click SET to apply **upper SET button in the Window position and size box**
→ Align Window: H.Center
  • Centers the window horizontally; adjust Left/Right if ShaderGlass covers 86Box.
→ Optional Scaling
  • 16x9 @ 4K → Width: 3853; Height: 2198
→ Save profiles for convenience
  
  
⎯(ᶜᵒʷ•ᵔ)⎯─◐◑◒◓─ Coolsoft VirtualMIDISynth
Description: Host-side MIDI synthesizer with customizable soundfonts.
→ Launch VirtualMIDISynth → Add/Select soundfont → Apply
→ Suggested soundfonts:
  • For elevated accuracy: Arachno SoundFont - Version 1.0.sf2
  • For dreamy and atmospheric: GeneralUser GS v1.471.sf2
  • For SC‑55‑like correctness.: Unison.SF2
  
  
⎯(ᶜᵒʷ•ᵔ)⎯─◐◑◒◓─ dgVoodoo2
Description: Wrapper that converts DirectX 1–7 and Glide calls into Direct3D 11/12 so legacy games run correctly. Fixes broken rendering, missing textures, color corruption, and scaling issues.
• Install dgVoodoo (latest version)
  → Copy required files to game root folder (not all files are necessary for all games; copying all files is harmless)
    • Control panel: dgVoodoo.conf, dgVoodooCpl.exe
    • DirectX 32 bit: MS\x86\D3D8.dll, MS\x86\D3D9.dll, MS\x86\D3DImm.dll, MS\x86\DDraw.dll
    • DirectX 64 bit: MS\x64\D3D9.dll   
    • Glide 32 bit: 3Dfx\x86\Glide.dll, 3Dfx\x86\Glide2x.dll, 3Dfx\x86\Glide3x.dll
    • Glide 64 bit: 3Dfx\x64\Glide.dll, 3Dfx\x64\Glide2x.dll, 3Dfx\x64\Glide3x.dll
• Configuration 
  → Launce dgVoodooCpl.exe from the game folder
  → DirectX
    • Application controlled fullscreen: Disabled
    • Disable Alt‑Enter: Enabled
    • dgVoodoo Watermark: Disabled
• Note: To access extended menus → right-click dgVoodoo Control Panel window → Show all sections of the configuration: Enabled  

  
⎯(ᶜᵒʷ•ᵔ)⎯─◐◑◒◓─ DXWND
Description: API‑hooking tool that forces windowed mode, fixes mouse capture, corrects timing, and applies compatibility patches for older Windows games.
  
  
⎯(ᶜᵒʷ•ᵔ)⎯─◐◑◒◓─ Lame Patcher
Description: Applies small binary .lpatch fixes to game executables
• Apply .lpatch to executable
  → Copy lpatch.exe to the game folder
  → Copy <game>.lpatch to the game folder
  → Launch lpatch.exe → Select <game>.lpatch → Select <file to patch>
  
  
⎯(ᶜᵒʷ•ᵔ)⎯─◐◑◒◓─ dinputto8
Description: Improves compatibility in games using DirectInput 1-7 (dinput.dll) 
• Copy the dinput.dll file into the folder of a DirectInput 1-7 program or game  
  
  
⎯(ᶜᵒʷ•ᵔ)⎯─◐◑◒◓─ DDrawCompat
Description: Fixes compatibility and performance issues with the DirectX 1-7 graphics APIs
• Copy ddraw.dll to the game install folder where the main executable (.exe) file resides
  
  
⎯(ᶜᵒʷ•ᵔ)⎯─◐◑◒◓─ K‑Lite Codec Pack
Description: Comprehensive pack of video and audio codecs. Available in four variants: Basic, Standard, Full, and Mega  
• Recommended variant: Standard — safe and sufficient for all pasture games
