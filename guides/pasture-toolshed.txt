Description: Tools for pacing, scaling, shading, MIDI rendering and compatibility. 
Note: Tools in this paddock are path‑specific. 
      Scaling tools (ShaderGlass, Lossless Scaling, SRWE) affect only the picture. 
      Audio tools (VirtualMIDISynth, SoundFonts) affect only MIDI. 
	  Wrappers (dgVoodoo2, DXWND) affect compatibility. 
      Lossless Scaling and ShaderGlass do not work well together. 
      Use SRWE for custom scaling when the other options (ShaderGlass, Lossless Scaling) are not adequate.
	  RTSS is optional and used only to cap the framerate.

⎯(ᶜᵒʷ•ᵔ)⎯─◐◑◒◓─ LOSSLESS SCALING
Description: Lossless Scaling settings optimized for VGA gaming at 60Hz **not compatible with Shaderglass**
→ Frame Generation → Type: LSFG 3.1 → Mode: Fixed → Multiplier: 4 → Flow scale: 55% → Performance: ON **preserves VGA detail**
→ Capture → Capture API: WGC; Queue target: 0
→ Scaling → Type: Sharp Bilinear **preserves VGA detail**
→ Rendering → Sync mode: Vsync; Max frame latency: 1; G-Sync support: On

⎯(ᶜᵒʷ•ᵔ)⎯─◐◑◒◓─ SHADERGLASS
Description: Applies .slang shaders in real time. **not compatible with Lossless Scaling.**
→ Launch ShaderGlass → Input → Window → select your game.  
→ Input → Pixel Size → x1
→ Shader → Choose From Library / Import Custom → choose shader. 
→ Output → Scale →  Scaling % **for custom scaling select FREE and use SRWE to adjust Shaderglass window**
  • Press 'm' to toggle menu on/off, useful when scaling with SRWE.
→ Performance tweaks **Undo if tearing presents in game**
  • Processing → Advanced → Use Flip Mode: Enabled; Allow Tearing: Enabled; Max Capture Rate: Enabled
  • Processing → FPS → 100% (VSync)
  • Hide menu bar in Shaderglass (Shortcut M) → Processing → Toggle Menu
→ Shader suggestions:
  • RetroArch Library/pixel-art-scaling/sharp-bilinear **purity and edge integrity**
  • misc/simple_color_control **faithfully enhanced color depth**

⎯(ᶜᵒʷ•ᵔ)⎯─◐◑◒◓─ SRWE (SIMPLE RUNTIME WINDOW EDITOR)
Description: Tool for resizing, repositioning, and border‑taming stubborn apps and games.
→ Launch SRWE → Select Running Application
  • Select Shaderglass.exe when using Shaderglass
→ Window position and size **Scale to 4K resolution at 4x3 aspect ratio** 
  **Press ENTER after entering each value or it won't apply**
  • Shaderglass Config **primary path**:
    → Y: -31; Width: 2896; Height: 2199 **press ENTER after each value**
  • 86Box Config **use only when not using ShaderGlass or LS**
    → Y: -53; Width: 2961; Height: 2221 **press ENTER after each value** 
→ Click SET to apply **upper SET button in the Window position and size box**
→ Align Window: H.Center
  • Centers the window horizontally; adjust Left/Right if ShaderGlass covers 86Box.
→ Optional Scaling
  • 16x9 @ 4K → Width: 3853; Height: 2198
→ Save profiles for convenience

⎯(ᶜᵒʷ•ᵔ)⎯─◐◑◒◓─ Coolsoft VirtualMIDISynth

  • For SC‑55‑like correctness.: Unison.SF2

⎯(ᶜᵒʷ•ᵔ)⎯─◐◑◒◓─ dgVoodoo2
Description: Wrapper that converts DirectX 1–7 and Glide calls into Direct3D 11/12 so legacy games run correctly. Fixes broken rendering, missing textures, color corruption, and scaling issues.
• Install dgVoodoo (latest version)
  → Copy required files to game root folder (not all files are necessary for all games; copying all files is harmless)
    • Control panel: dgVoodoo.conf, dgVoodooCpl.exe
    • DirectX 32 bit: MS\x86\D3D8.dll, MS\x86\D3D9.dll, MS\x86\D3DImm.dll, MS\x86\DDraw.dll
    • DirectX 64 bit: MS\x64\D3D9.dll   
    • Glide 32 bit: 3Dfx\x86\Glide.dll, 3Dfx\x86\Glide2x.dll, 3Dfx\x86\Glide3x.dll
    • Glide 64 bit: 3Dfx\x64\Glide.dll, 3Dfx\x64\Glide2x.dll, 3Dfx\x64\Glide3x.dll
• Configuration 
  → DirectX
    • Application controlled fullscreen: Disabled
    • Disable Alt‑Enter: Enabled
    • dgVoodoo Watermark: Disabled

⎯(ᶜᵒʷ•ᵔ)⎯─◐◑◒◓─ DXWND
Description: API‑hooking tool that forces windowed mode, fixes mouse capture, corrects timing, and applies compatibility patches for older Windows games.
