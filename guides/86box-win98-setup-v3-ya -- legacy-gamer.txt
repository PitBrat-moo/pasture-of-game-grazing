; Verify the display index before importing.

⟦⟦⟦~⟡🐄⟧⟧⟧ QUICK TROUBLESHOOTING ⟦⟦🐄⟣~⟧⟧⟧
Symptom | Likely cause | First action
⟢ FMV drops / stutter | Per‑game FMV timing sensitivity | Set MHz per‑game: FMV‑heavy titles prefer lower MHz; 3D‑heavy titles prefer higher MHz
  • Optional: lower CD‑ROM speed (24×/12×) for disc-image read consistency

Description: Determine optimum CPU frequency (MHz) settings on a per‑game basis.  
→ Check recommended system requirements for the title.  
  • FMV‑heavy titles: ~100–133 MHz
  • Dialogue-heavy & 3D/2D hybrid titles: ~233–300 MHz
  • 3D Accelerated titles: ~333–366 MHz
  • Late era titles (2000–2002): ~366–400 MHz
→ Run three titles:  
  • FMV / Dialogue / MIDI heavy title — listen for audio jitter and note timing.
  • Redbook / CD audio title — verify track switching and volume.
  • 3D Accelerated title (Glide, D3D, OpenGL MiniGL) — note emulation speed dips, frame pacing, and jitter.  
→ If jitter, reduce MHz and retest.
→ Log results: title, tuned frequency.
