## Notes  

- Temujin streams directly from CD‑ROM; no installation is performed.  
- Requires a Win98SE VM for correct FMV playback and timing.  
- CPU frequency should remain ≤300 MHz for stability.  
- Uses native Win98 multimedia stack; wrappers are not required.  
- Save files are stored in **C:\Temujin\SavedGames** inside the VM.  

⚠️ Stable under 98SE in 86Box  
Avoid modern Windows execution; FMV and timing issues are severe.
