## Notes  

- Install outside Program Files to avoid config/save permission issues.  
- Do **not** install the VP3 codec; use K‑Lite if video playback is desired.  
- Patch 1.09 is the latest English version; 1.10 is German‑only.  
- dgVoodoo provides the most stable renderer; do not combine with DXWND.  
- Weather Effects must be disabled to prevent VRAM leak and lighting bugs.  
- Runs best at a strict 60 FPS cap (critical for internal timing).  

⚠️ Stable under Windows 10/11 with dgVoodoo  
Avoid mixing wrappers; use dgVoodoo alone for correct behavior.
